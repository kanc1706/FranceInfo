# FranceInfo Web Aplikacija

Ovaj projekt predstavlja završnu fazu izrade web aplikacije za vijesti, razvijenu u sklopu kolegija Web i XML te PWA.

## Funkcionalnosti
* Prikaz vijesti
* Prikaz vijesti po kategorijama
* Administracija vijesti (CRUD)
* Registracija korisnika
* Prijava i odjava korisnika
* Administratorske ovlasti
* Vremenska prognoza za francuske gradove (tečajna lista integrirana kroz API)
* REST API integracija
* JSON obrada podataka

## Korištene tehnologije
* PHP
* MySQL
* HTML5
* CSS3
* REST API 
* JSON

## Pokretanje projekta
1. Kopirati projekt u `htdocs` direktorij XAMPP-a.
2. Pokrenuti Apache i MySQL servis u XAMPP Control Panelu.
3. U phpMyAdmin-u kreirati bazu podataka.
4. Importati priloženu datoteku `franceinfo.sql`.
5. Otvoriti aplikaciju u pregledniku na adresi:
   `http://localhost/Web_i_XML_Projekt/index.php`
